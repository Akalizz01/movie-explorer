# 🎬 Movie Web App

A full-stack movie web application that allows users to browse movies using the TMDB API, create accounts, log in, save favorite movies, create custom lists, and write reviews.

The platform provides an interactive experience where users can explore movies, manage their personal collections, and share their opinions through reviews.

---

# 🚀 Features

- 🔐 User authentication system (Register / Login / Logout)
- 🎥 Browse movies using the **TMDB API**
- 🔎 Search for movies
- 📄 View detailed movie information
- ❤️ Add or remove movies from **favorites**
- 📋 Create and manage **custom movie lists**
- ✍️ Write and manage **movie reviews**
- ⭐ Rate movies and share opinions
- ➕ Add movies to personal lists
- ❌ Remove movies from lists
- 💾 Store user data in a **MySQL database**
- 🖥️ Interactive and user-friendly interface

# 🏗️ Project Architecture

The application is divided into two main parts:

**Frontend**
- Handles the user interface
- Displays movies from the TMDB API
- Manages user interactions

**Backend**
- Handles authentication
- Communicates with the database
- Manages user data, favorites, and lists

**Database**
- Stores user accounts
- Stores favorite movies
- Stores user-created movie lists

# ⚙️ Installation


## 1 Clone the repository
```bash
git clone https://github.com/Akalizz01/movie-explorer.gi
```
## 2 Go to the project folder
```bash
cd movie-explorer
```
## 3 Install dependencies
```bash
npm install
```
## 4 Configure the database
Go to the directory backoffice/recursos/database/db.js and configure it according to your database's needs.
## 5 Start the server
```bash
cd backoffice
node server.js
```
